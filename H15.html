Installation Guide

The H15 Series features true plug-and-play installation specifically designed for H15 socket vehicles.

Step-by-Step Installation

  1. Disconnect the vehicle battery for safety
  2. Remove the old H15 halogen bulbs
  3. Install the H15 Series LED bulbs
  4. Position the turbo charged fan for optimal airflow
  5. Ensure secure H15 socket connection
  6. Adjust the beam pattern and test the lights

Installation Tips

  • Verify H15 compatibility before installation
  • Ensure proper fan clearance for optimal cooling
  • Check beam alignment after installation
  • The turbo fan operates automatically
