Installation Guide

Easy plug-and-play installation. Simply remove your old halogen bulbs and replace them with the Y2 Series LED headlights.

Step-by-Step Installation

  1. Disconnect the vehicle battery
  2. Remove the old headlight bulbs
  3. Install the Y2 Series LED bulbs
  4. Adjust the beam pattern if necessary
  5. Reconnect the battery and test the lights

  • Handle bulbs with clean hands or gloves
  • Check beam alignment after installation
