The D90 Series features true plug-and-play installation with intelligent driver technology that eliminates compatibility issues.

  1. Disconnect the vehicle battery for safety
  2. Remove the old headlight bulbs from their sockets
  3. Install the D90 Series LED bulbs - the intelligent driver ensures proper connection
  4. Ensure the cooling system has adequate airflow
  5. Adjust the beam pattern for optimal road coverage
  6. Reconnect the battery and test the lights
  7. No error code configuration required - the intelligent driver handles compatibility
