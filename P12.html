Installation Guide

The P12 Series features professional-grade installation with advanced cooling system integration. While still plug-and-play compatible, optimal performance requires proper cooling system setup.

Step-by-Step Installation

  1. Disconnect the vehicle battery for safety
  2. Remove the old headlight bulbs from their sockets
  3. Install the P12 Series LED bulbs - ensure proper alignment
  4. Position the triple copper tube cooling system for optimal airflow
  5. Secure all connections and ensure no obstructions to cooling tubes
  6. Adjust the beam pattern for optimal road coverage
  7. Reconnect the battery and test the lights

Installation Tips

  • Ensure adequate clearance around all cooling tubes
  • Avoid contact with other engine components for optimal heat dissipation
  • Test cooling system performance after installation
